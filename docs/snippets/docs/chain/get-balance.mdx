### getBalance

Gets the native token balance of an address in the chain's base units.

```ts twoslash
// [!include ~/snippets/code/chain.ts]
// ---cut---
const balance = await chain.getBalance('nativeAddress')
```

**Parameters:**

- `address`: The {props.chainName} address to check

**Returns:**

- The balance in the chain's base units as a bigint
